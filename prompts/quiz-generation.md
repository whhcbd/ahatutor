# 出题 Prompt 模板

你是一位遗传学教师，请根据以下知识点出一道选择题：

**知识点**: {topic}
**难度**: {difficulty}
**用户水平**: {userLevel}
**历史错题**: {mistakes}

## 要求

1. 题目要考查核心概念
2. 4个选项，只有一个正确
3. 不要涉及超纲内容
4. 返回 JSON 格式

## 返回格式

```json
{
  "question": "题目内容",
  "options": [
    {"id": "A", "label": "A", "content": "选项内容"},
    {"id": "B", "label": "B", "content": "选项内容"},
    {"id": "C", "label": "C", "content": "选项内容"},
    {"id": "D", "label": "D", "content": "选项内容"}
  ],
  "correctAnswer": "A",
  "explanation": {
    "level1": "最简练解析（如：3:1）",
    "level2": "简要解释（如：显性:隐性 = 3:1）",
    "level3": "标准解析（包含基因型、表型、比例的完整说明）",
    "level4": "详细讲解（添加原理说明和注意事项）",
    "level5": "教学级解析（从基础概念开始，完整推导过程）"
  },
  "topic": "知识点",
  "tags": ["标签1", "标签2"]
}
```

## 示例

**知识点**: 孟德尔第一定律
**难度**: medium
**用户水平**: intermediate

**输出**:

    {"id": "B", "label": "B", "content": "1:1"},
    {"id": "C", "label": "C", "content": "1:2:1"},
    {"id": "D", "label": "D", "content": "9:3:3:1"}
  ],
  "correctAnswer": "A",
  "explanation": {
    "level1": "3:1",
    "level2": "高茎:矮茎 = 3:1",

    "level4": "根据孟德尔分离定律，杂合子Dd自交，配子D和d各占1/2。随机结合后，基因型比例为DD:Dd:dd = 1:2:1。由于D对d为显性，DD和Dd都表现为高茎，dd表现为矮茎，所以表现型比例为3:1。",

  },
  "topic": "孟德尔第一定律",
  "tags": ["孟德尔定律", "分离定律", "基因型", "表型"]
